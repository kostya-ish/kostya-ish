## Привет, я Костя! 👋

Junior QA Engineer с практическим опытом ручного тестирования веб и мобильных приложений на командных проектах.

До QA - 9 лет в управлении ресторанами (команды до 50 человек, выручка до 22 млн ₽/мес). Этот бэкграунд дал мне то, что в QA называется системным мышлением: я привык писать регламенты и чек-листы, искать корневые причины сбоев, работать с обратной связью как с потоком багов и измерять результат метриками (рейтинг, food cost, скорость онбординга).

Переход в IT - осознанный и подкрепленный: расширенный курс Яндекс Практикума, 2 командных проекта, продолжаю изучать Stepik, Хабр, Куликова, погружаюсь в Kafka

### Технологии и инструменты:
- Тестирование: функциональное, исследовательское, регрессионное, интеграционное, базовые нефункциональные проверки
- Документация и дефекты: чек-листы, тест-кейсы, тест-планы, баг-репорты, ретест/верификация
- Web/UI: DevTools, кроссбраузерное/кроссплатформенное тестирование, проверка по макетам Figma
- API: Postman, Swagger (валидация ответов, проверки вложенных объектов, URL-параметров, граничных значений)
- Данные: SQL (SELECT, WHERE, COUNT, DISTINCT, JOIN)
- Сеть: Charles Proxy
- Mobile: Android Studio (эмуляторы)

### Стек:
TestIT, Yandex Tracker, Postman, Swagger, JSON Schema, Kibana, DBeaver, SQL (PostgreSQL), Charles Proxy, Figma, Android Studio, BrowserStack, DevTools, Python, pytest.
